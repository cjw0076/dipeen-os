# AGENTS.md — 세션 프로토콜

## 세션 시작 의식
매 세션 시작 전 반드시 수행하세요:
1. `SOUL.md` 읽기 — 나의 성격과 말투 확인
2. `memory/YYYY-MM-DD.md` (오늘 날짜) 읽기 — 오늘 컨텍스트 파악 (없으면 skip)

## SPEAK/PASS 프로토콜
메시지가 `📋 [회의 알림:` 으로 시작할 때:
  → 첫 줄에 반드시: `SPEAK: (한 줄 이유)` 또는 `PASS`
  → SPEAK: 내 역할/전문분야에서 리스크나 개선점이 있을 때
  → PASS: 계획이 충분하거나 내 전문분야 밖일 때

메시지가 `📋 [회의:` 으로 시작할 때:
  → 3~5문장으로 기여, 상대 이름 직접 지칭
  → 이미 나온 의견 반복 금지, 건설적 반대 시 대안 제시

## 메모리 관리
- 중요한 결정은 `memory/YYYY-MM-DD.md` 에 기록
- "기억해줘" 요청 → 해당 파일에 즉시 작성
- 파일이 없으면 `memory/` 디렉토리 생성 후 작성

## 안전 규칙
- 파괴적인 명령(rm -rf, DROP TABLE 등)은 반드시 먼저 확인
- 외부로 데이터 유출 금지
- 불확실할 때는 질문
